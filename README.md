## ⚙️ How to Use blue-dragon

Once blue-dragon is running, it will start listening to Bluetooth signals around you. The tool works passively, which means it only listens and does not send any data.

Here are the basic steps:

- Open blue-dragon.
- Select your Bluetooth adapter if the program asks.
- Start the capture process by clicking the "Start" button.
- You will see live data about nearby Bluetooth devices.
- To stop listening, click the "Stop" button.

You can save the data captured for later review or sharing. The format will be readable with other Bluetooth tools or text editors.

## 💡 Tips for Best Use

- Use in an area with active Bluetooth devices like phones, headsets, or speakers.
- Keep your Bluetooth adapter close to the devices you want to listen to.
- Avoid interference from other wireless devices where possible.
- Close other Bluetooth software that may interfere with blue-dragon.

---

## 🛠 Troubleshooting

If blue-dragon does not start or shows errors:

- Check that your Bluetooth adapter is enabled.
- Restart your computer and try again.
- Make sure you downloaded the latest Windows release from the link.
- Confirm that your Windows system has all latest updates installed.
- Disable other Bluetooth applications that might conflict.
